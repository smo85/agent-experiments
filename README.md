# FiveTodo

## Product
FiveTodo is a simple web-based todo app that limits users to at most five active todos.

## Rules
- Max 5 active todos
- Fields: title, priority (low|medium|high), due_date, completed

## Stack
- Backend: FastAPI
- DB: SQLite
- Tests: pytest

## Workflow
- Work is done via GitHub issues
- One issue per PR
- Tests required
